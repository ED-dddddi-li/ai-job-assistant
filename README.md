# 🤖 AI 求职助手

> 输入岗位 JD + 简历，自动完成岗位分析、简历优化、面试题生成

**🔗 在线体验：[https://ed-dddddi-li.github.io/ai-job-assistant/](https://ed-dddddi-li.github.io/ai-job-assistant/)**

**技术栈**：HTML · CSS · JavaScript · DeepSeek API

---

## ✨ 功能介绍

### 🔍 岗位分析
- 解析 JD 核心职责、必备技能、隐藏要求
- 结合个人简历给出匹配度评分（0-100分）
- 提供明确的投递建议和准备方向

### 📝 简历优化
- 从简历中找出最匹配岗位的项目经历并排序
- 逐条给出改写建议，用数据和关键词强化描述
- 给出结构调整和关键词补充建议

### 🎯 面试备考
- 根据 JD + 简历生成最可能被问到的面试题
- 分三类：专业技能题 / 项目经历题 / 综合压力题
- 每题附考察点 + 个性化参考答案

---

## 🚀 使用方法

1. 打开 [在线地址](https://ed-dddddi-li.github.io/ai-job-assistant/)
2. 填入 [DeepSeek API Key](https://platform.deepseek.com)（注册即送免费额度）
3. 粘贴简历内容（填一次，三个功能共用）
4. 选择功能标签，粘贴目标岗位 JD，点击分析

> 本工具完全在本地运行，API Key 不会上传至任何服务器
